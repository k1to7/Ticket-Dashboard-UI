## <a id="english"></a> 🇺🇸 English Version

🔴 **[Live Demo: Click here to access the application]([COLE SEU LINK DA VERCEL AQUI])**

This project is a modern, responsive Front-End dashboard built to manage service requests and track SLAs. It serves as the visual interface (UI/UX) for my automated triage and ticket management workflow portfolio.

### Tech Stack
* **HTML5:** Semantic markup structure.
* **Tailwind CSS:** Utility-first styling to build a clean, corporate SaaS-level design without relying on heavy frameworks.
* **Bilingual Architecture:** Native page toggle between English and Portuguese environments.

### ✨ Features
* **Metrics Overview:** KPI cards highlighting Open Tickets, Overdue SLAs, and recently Resolved demands.
* **Ticket Tracking:** Dynamic data table with status badges and priority tracking.
* **Responsive Design:** Fully adaptable layout for seamless use on desktop and mobile devices.

## <a id="português"></a> 🇧🇷 Versão em Português

**[Live Demo: Clique aqui para acessar a aplicação](https://ticket-dashboard-ui-eight.vercel.app/)**

Este projeto é um painel Front-End moderno e responsivo criado para o gerenciamento de chamados e rastreamento de prazos (SLAs). Ele atua como a interface visual (UI/UX) do meu portfólio de triagem automatizada e gestão de tickets.

### Tecnologias Utilizadas
* **HTML5:** Estruturação semântica.
* **Tailwind CSS:** Estilização baseada em classes utilitárias para garantir um design corporativo (SaaS) limpo, moderno e rápido.
* **Suporte Bilíngue:** Alternância nativa de interface entre Português e Inglês.

### Funcionalidades
* **Visão Geral de Métricas:** Cartões de KPI destacando Chamados Abertos, SLAs Atrasados e demandas Resolvidas.
* **Gestão de Tickets:** Tabela dinâmica de chamados com indicadores visuais de status e prioridade.
* **Design Responsivo:** Layout 100% adaptável para uso em computadores e dispositivos móveis.
